Getting Started

Include the libraries in the document head 
            
            <head> 

                <title>Your Title</title> 

                <!-- First socket.io.js , then lib.js in the libraries folder--> 

                <script src="../libraries/socket.io.js"></script> 

                <script src="../libraries/lib.js"></script> 

            </head>

Start

Use controls.start function 
            
            controls.start(
                ()=>{ start_the_game() }
            )
Suppose foo and bar are using controls 
            
            controls.start(
                players=>{ console.log(players); // logs ["foo" , "bar"] }
            )

Press_Start Event

This event is fired when a key is pressed 
            
            controls.press_start(
                ()=>{ function() }
            )
Suppose a key is pressed by user foo 
            
            controls.press_start(
                (event)=>{ 
                    console.log(event)
                    // logs either on the basis of the buton pressed:
                    {e:"ps" , key:"Up" , code:85 , id:"foo"}
                    {e:"ps" , key:"Left" , code:76 , id:"foo"}
                    {e:"ps" , key:"Right" , code:82 , id:"foo"}
                    {e:"ps" , key:"Down" , code:68 , id:"foo"}
                    {e:"ps" , key:"Y" , code:89 , id:"foo"}
                    {e:"ps" , key:"X" , code:88 , id:"foo"}
                    {e:"ps" , key:"A" , code:65 , id:"foo"}
                    {e:"ps" , key:"B" , code:66 , id:"foo"}
                 }
            )

Press_End Event

This event is fired when a key is released 
            
            controls.press_end(
                ()=>{ function() }
